## Project CodeChain

Le projet CodeChain a pour but de fournir à BeCode une plateforme d'apprentissage similaire
à Codecademy. Cette plateforme permettrait à BeCode d'être totalement indépendante et de fournir
un outil de formation/d'évaluation aux  futurs apprenants.



## codemirror

Codemirror a été choisi pour devenir l'IDE virtuel de CodeChain. Il permet d'écrire dans presque tous les
langages, possède une fonction d'auto-complétion et même de personnaliser son thème.
